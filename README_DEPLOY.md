# SecureMailer Landing

Static landing page: no build, no backend, no npm.

## Before publishing

Open `site.config.js` and set one of these:

```js
contactEmail: "your@email.com"
```

or:

```js
contactUrl: "https://t.me/your_username"
```

## Local preview

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```
